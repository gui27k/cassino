# 🎰 Royal Vegas Casino

Um cassino virtual completo desenvolvido em **HTML, CSS e JavaScript puro**, com interface moderna inspirada em plataformas de jogos online.

## ✨ Visão Geral

Royal Vegas Casino é uma aplicação web single-page que reúne diversos jogos clássicos de cassino em uma única interface responsiva. O projeto foi desenvolvido sem frameworks externos, utilizando apenas tecnologias nativas da web.

## 🎮 Jogos Disponíveis

### ♠ Blackjack
- Sistema completo de cartas
- Dealer automático
- Cálculo de pontuação
- Regras clássicas de Blackjack
- Controle de apostas

### 🎰 Slots 3x3
- Grade 3x3 de símbolos
- Múltiplas linhas vencedoras
- Sistema de jackpot
- Recompensas automáticas

### 🎡 Roulette
- Roleta animada
- Apostas em vermelho ou preto
- Sistema de rotação visual
- Resultado aleatório

### 💣 Mines
- Campo com 25 posições
- Quantidade de bombas configurável
- Multiplicador progressivo
- Sistema de saque antecipado (Cashout)

### ✈️ Aviator (Crash)
- Multiplicador crescente em tempo real
- Sistema de cashout
- Ponto de queda aleatório
- Mecânica inspirada em jogos Crash
